# 推しのYoutubeのチャンネルにある歌配信の中から指定のお歌をすぐに見つけたい!

### 心の叫び
Youtubeで配信者として活動している"推し"はめちゃくちゃ歌がうまい！！活動の中で配信で歌配信をすることもある。我々、リスナーとしては配信後、"あの配信で歌ってた曲をもう一度聞きたい"と思うのはよくあることである。</br>
しかし！！歌枠がいっぱいあってお歌を探すことができない！！</br>
ここではこれを解決する。

### 使用するもの
- Python3.9
- Heroku
- Line API
- PostgreSQL
- Google Sheet API
