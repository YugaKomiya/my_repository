# Twitter bot を作成しました

### 概要
私の推したちのアカウントを1分間隔でチェックして、プロフィールに変更があればツイートして通知するbotです。</br>
2021/3/1 アカウント名やbio、アイコンやヘッダーの変更を通知できるように変更。

### 使用したもの
- Python3.9
- Heroku
- Twitter API
- Google Sheet API
- Google Drive API
